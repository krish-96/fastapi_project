### Tips To reduce the token size and control the response size

Since you're a backend engineer, here are a few other shorthand prompts that work well:

* TL;DR → Short summary only.
* ELI5 → Explain Like I'm 5 (very simple explanation).
* Deep dive → Detailed explanation.
* Code only → Just the code, no explanation.
* Pros/Cons → Compare advantages and disadvantages.
* Table → Present the answer as a table.
* Example → Focus on a practical example.
* Step-by-step → Explain the process in sequence.
